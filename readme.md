# eslint-config

## Usage

```sh
$ npm i -D eslint eslint-config-smashcast-angular
```

Then add it to your .eslintrc

```
{
  "extends": [
    "smashcast-angular"
  ]
}
```


To get an insight how it works check the [official documentation](http://eslint.org/docs/developer-guide/shareable-configs).
